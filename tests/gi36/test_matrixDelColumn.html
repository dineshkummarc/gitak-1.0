Matrix Delete Column test, make sure record deletion works with delete column. Should be tested over HTTPS.
Test Matrix delete column
include testSetup.html  
jsxopen ${GIPATH}/shell.html?jsxapppath=${SAMPLESPATH}/test-matrix  
waitForElementPresent JsxToolbarButtonName=tbbList 5000
clickJsxElement JsxToolbarButtonText=List  
waitForElementPresent JsxMatrixCellText=matrix1,Antarctica 5000
// Sort on delete column first. IE7 is reverse of other browsers
clickJsxMatrixHeader JsxMatrixHeaderIndex=matrix1,0  
pause 700  
assertElementPresent JsxMatrixCellIndex=matrix1.14.0 // last row is index 14 of 0-14
// Delete Angola
clickJsxCellMask JsxMatrixCellId=matrix1.AO.0  
verifyElementNotPresent JsxMatrixCellId=matrix1.AO.0  
assertElementNotPresent JsxMatrixCellIndex=matrix1.14.0 // row 14 was deleted
// Delete Azerbaijanen
verifyElementPresent JsxMatrixCellId=matrix1.AZ.0  
clickJsxCellMask JsxMatrixCellIndex=matrix1.13.0 //one row deleted only 0-13 left
verifyElementNotPresent JsxMatrixCellId=matrix1.AZ.0  
// Delete Albenia
verifyElementPresent JsxMatrixCellId=matrix1.AL.0  
clickJsxCellMask JsxMatrixCellId=matrix1.AL.0  
verifyElementNotPresent JsxMatrixCellId=matrix1.AL.0
